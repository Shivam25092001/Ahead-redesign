## Tech Stack

**Client:** NEXT.js
**Frameworks:** Framer Motion, Tailwind CSS
**Hosting Platform:** Vercel



## Installation - Front-end
1. Clone the repository:
```bash
  git clone https://github.com/Shivam25092001/Ahead-redesign
```

2. Navigate to the project directory:
```bash
  cd Ahead-redesign
```

3. Install dependencies using npm:
```bash
  npm install
```
